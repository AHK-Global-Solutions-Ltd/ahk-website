## üöÄ **Quick Start**

### **Prerequisites**
- Node.js 18+ 
- pnpm (recommended) or npm

### **Installation**
```bash
# Install dependencies
pnpm install

# Start development server
pnpm run dev

# Build for production
pnpm run build

# Preview production build
pnpm run preview
```

### **Development Server**
```bash
pnpm run dev
```
Open [http://localhost:5173](http://localhost:5173) to view in browser.
